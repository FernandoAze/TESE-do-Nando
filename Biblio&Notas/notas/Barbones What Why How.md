# What?
BeatScoreWarped is a framework designed for MIR-related visualizations that enables the visualization of data aligned with a spectrogram and a musical score. This tool combines matplotlib-based graphs and visualizations with a musical score that is temporally aligned with them. The output is an SVG file that can be further edited to generate custom visualizations.
 
# Why?
Although tools like matplotlib and  Sonic Visualizer (amoung many others) already have powerful capabilities for generating MIR-relevant visualizations, none combine musical scores in an aligned manner with other visual elements and graphs. This work serves primarily as a proof of concept for the value of aligned score visualization as a feature for MIR research.
Furthermore, beyond its use as a purely visual tool, this approach may offer benefits for interactive Beat Annotation and other MIR annotation and analysis tasks.
 
# How?
BeatScoreWarped combines plots generated by Matplotlib and aligns them to a .mei musical score on the same timeline. The time-aligned score (generated through ScoreWarp) is provided in .svg format. BeatScoreWarped then adds matplotlib graphs as vectorial (or rasterized) elements on top of that score. The result is a single .svg file with multiple layers that, due to their vectorial nature, can be repositioned to create desired visualizations.
